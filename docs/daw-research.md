# Open-Source DAW / Audio Harness Research

Research question:

> What open-source DAW or audio engine is the best target for adding code so Humforge can use the DAW itself as the agent-controlled harness?

Evaluation criteria:

- license compatibility
- active maintenance
- build complexity
- scripting/API/plugin surface
- clip/timeline model
- render/export support
- plugin hosting support
- agent-control friendliness
- macOS support
- ability to integrate with chat/worker processes

Candidates to investigate include:

- Ardour
- Zrythm
- LMMS
- Bespoke Synth
- Audacity / Tenacity
- Tracktion Engine / Waveform-related options
- Reaper via scripting, even though not open-source, as a practical comparison baseline
